# CUTBAR FINANCE

A next-generation crypto-powered social gaming ecosystem.

CUTBAR is not just a token — it is a system where:
- Users play
- Users earn
- Users interact in real-time

## Core Features

- 🎮 Chatroom-based games (LowCard, Last Standing Wins)
- 💰 Integrated wallet & token economy
- 🧠 Smart UI with real-time interaction
- 🎁 Tipping, gifting, raining system for XP growth
- 📊 Market, trading, and token insights

## Vision

To merge gaming, finance, and social interaction into a single ecosystem where engagement creates value.

## Tech Stack

- React (Frontend)
- Firebase (Backend services)
- Web3 Integration (Planned)
- BSC Network (Token)

## Roadmap

- Phase 1: UI + Core Pages ✅
- Phase 2: Chatroom Games Engine 🔄
- Phase 3: Token Integration ⏳
- Phase 4: Real Earnings System ⏳
- Phase 5: Global Launch 🚀
